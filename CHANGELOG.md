# 변경 이력 (CHANGELOG)

이 프로젝트의 주요 변경을 기록한다. 버전은 [Semantic Versioning](https://semver.org/lang/ko/)을 따른다.

## 0.1.0 (2026-07-05)

첫 공개 릴리스. 한국 근로기준법 기반 계산을 순수 함수 + 테스트로 제공한다.

### 추가된 모듈

- **overtime** — 근로기준법 제56조 배율(연장·야간·휴일)을 분 단위로 적용하는 수당 계산.
  주말출장 최소금액 보장, 회사별 취업규칙 옵션(`saturdayType`, `nightRange`) 지원.
- **leave** — 연차 발생·가산·주기 이월 차감, 보건휴가 공제(순수 계산, DB 비종속).
- **holidays** — 한국천문연구원(KASI) 특일정보 API 클라이언트 + 메모리 캐시 + 대체공휴일 +
  내장 폴백 표(`HolidayClient`).
- **payroll** — 월급날·연장근로 마감일·집계기간 계산. 공휴일 판정은 콜백 주입.
- **certificate** — 재직증명서 A4 HTML 빌더(프레임워크 무관), 날짜 포맷·주민번호 마스킹,
  발급번호 채번 순수 함수.

### 특징

- 런타임 의존 패키지 0개(계산은 전부 순수 함수, 공휴일 조회만 전역 `fetch` 사용).
- React/Next.js/DB 비종속 — 값 입력 → 값 출력. 필요한 `src/<모듈>/` 폴더만 복사해도 독립 동작.
- 타입스크립트 strict, 4개 타임존(UTC/KST/양수·음수 오프셋)에서 동일 결과 검증.
- 테스트 90개 통과, `examples/`의 실행 가능한 예제 4종 포함.
- MIT 라이선스.
